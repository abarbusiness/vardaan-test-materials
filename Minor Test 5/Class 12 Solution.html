DETAILED EXPLANATIONS
1. Answer: (a) [-π/2, π/2]
Explanation: By definition, the restricted domain of the sine function that makes it one-to-one and onto is [-π/2, π/2]. Therefore, the principal value branch (range) of its inverse, sin-1x, is [-π/2, π/2].
2. Answer: (c) [-1, 1]
Explanation: The domain of an inverse trigonometric function is the range of its corresponding basic trigonometric function. Since the range of f(x) = cos(x) is [-1, 1], the domain of cos-1x is [-1, 1].
3. Answer: (b) π/6
Explanation: Let sin-1(1/2) = y. Then sin(y) = 1/2. We know that sin(π/6) = 1/2 and π/6 lies in the principal value branch [-π/2, π/2]. Hence, the principal value is π/6.
4. Answer: (a) π/3
Explanation: Let cos-1(1/2) = y. Then cos(y) = 1/2. We know that cos(π/3) = 1/2 and π/3 lies in the principal value branch [0, π]. Hence, the principal value is π/3.
5. Answer: (d) π/4
Explanation: Let tan-1(1) = y. Then tan(y) = 1. We know that tan(π/4) = 1 and π/4 lies in the principal value branch (-π/2, π/2). Hence, the principal value is π/4.
6. Answer: (b) (-π/2, π/2)
Explanation: By standard definition, the principal value branch of tan-1x is the open interval (-π/2, π/2) because tan(x) is not defined at -π/2 and π/2.
7. Answer: (a) π/3
Explanation: We know the property sin-1(sin x) = x, provided x ∈ [-π/2, π/2]. Since π/3 is inside this interval, sin-1(sin(π/3)) = π/3.
8. Answer: (c) π - cos-1x
Explanation: This is a standard identity for inverse trigonometric functions. For x ∈ [-1, 1], cos-1(-x) = π - cos-1x. This ensures the output remains in the principal branch [0, π].
9. Answer: (a) -tan-1x
Explanation: This is a standard identity. For x ∈ R, the function tan-1x is an odd function, so tan-1(-x) = -tan-1x.
10. Answer: (b) π/2
Explanation: This is a fundamental complementary angle identity in inverse trigonometry: sin-1x + cos-1x = π/2 for all x ∈ [-1, 1].
11. Answer: (a) π/2
Explanation: This is another fundamental complementary angle identity: tan-1x + cot-1x = π/2 for all real numbers x ∈ R.
12. Answer: (c) π/3
Explanation: Let sec-1(2) = y. Then sec(y) = 2, which means cos(y) = 1/2. We know cos(π/3) = 1/2. The value π/3 lies in the principal value branch [0, π] - {π/2}.
13. Answer: (d) R - (-1, 1)
Explanation: sec(x) takes values in the range (-∞, -1] ∪ [1, ∞). Therefore, the domain of sec-1x is R - (-1, 1), which is the same as x ≤ -1 or x ≥ 1.
14. Answer: (b) π/6
Explanation: Let cosec-1(2) = y. Then cosec(y) = 2, meaning sin(y) = 1/2. The value π/6 lies in the principal branch [-π/2, π/2] - {0}. Thus, the answer is π/6.
15. Answer: (c) 2π/3
Explanation: Using the property cos-1(-x) = π - cos-1x, we get cos-1(-1/2) = π - cos-1(1/2) = π - π/3 = 2π/3.
16. Answer: (d) 1
Explanation: First evaluate sin-1(-1/2) = -π/6. Substitute this back: sin(π/3 - (-π/6)) = sin(π/3 + π/6) = sin(π/2) = 1.
17. Answer: (a) -π/3
Explanation: tan-1(√3) = π/3.
sec-1(-2) = π - sec-1(2) = π - π/3 = 2π/3.
Expression = π/3 - 2π/3 = -π/3.
18. Answer: (b) π/3
Explanation: We cannot directly write 2π/3 because it is outside the principal branch [-π/2, π/2]. Instead, sin(2π/3) = sin(π - π/3) = sin(π/3). Now, sin-1(sin(π/3)) = π/3.
19. Answer: (c) -π/4
Explanation: 3π/4 is outside the principal branch (-π/2, π/2) for tan-1. Write tan(3π/4) = tan(π - π/4) = -tan(π/4) = -1. Therefore, tan-1(-1) = -π/4.
20. Answer: (a) 0
Explanation: By the complementary angle identity, sec-1x + cosec-1x = π/2 for |x| ≥ 1. Therefore, cos(sec-1x + cosec-1x) = cos(π/2) = 0.
21. Answer: (b) [-1/2, 1/2]
Explanation: The domain of sin-1(y) is -1 ≤ y ≤ 1. Here y = 2x, so we have -1 ≤ 2x ≤ 1. Dividing by 2, we get -1/2 ≤ x ≤ 1/2. Hence the domain is [-1/2, 1/2].
22. Answer: (c) [0, π] - {π/2}
Explanation: By definition, the principal value branch of sec-1x is [0, π] excluding π/2 because sec(π/2) is undefined.
23. Answer: (a) 4/5
Explanation: Let cos-1(3/5) = θ. Then cos(θ) = 3/5. By Pythagorean identity, sin(θ) = √(1 - cos2θ) = √(1 - 9/25) = √(16/25) = 4/5 (positive since θ is in quadrant I).
24. Answer: (d) 5π/6
Explanation: Using the property cot-1(-x) = π - cot-1x, we have cot-1(-√3) = π - cot-1(√3) = π - π/6 = 5π/6.
25. Answer: (b) 3π/4
Explanation: We know cos-1(-1/2) + sin-1(-1/2) = π/2 (using sin-1x + cos-1x = π/2).
Also, tan-1(1) = π/4.
Adding them together: π/4 + π/2 = 3π/4.
