Minor Test 04
Class: 8 CBSE Topic: Algebraic Expressions & Identities Time: 1 Hour Max. Marks: 40
General Instructions:
  • All questions are compulsory.
  • The paper is divided into three sections: A, B, and C.
  • Section A contains 5 questions of 2 marks each.
  • Section B contains 5 questions of 3 marks each.
  • Section C contains 3 questions of 5 marks each.
  • Show all working steps clearly and mention the identities used.
SECTION A (10 Marks)(Attempt all questions - 2 Marks Each)
1.
Identify the terms and their coefficients in the following algebraic expression:
$5xyz^2 - 3zy$
2.
Add the following algebraic expressions:
$ab - bc, \quad bc - ca, \quad ca - ab$
3.
Find the product of the monomial and the binomial:
$3x \times (4x - 5y)$
4.
Use a suitable identity to find the product:
$(2x + 3y)^2$
5.
Evaluate using an appropriate identity:
$51^2 - 49^2$
SECTION B (15 Marks)(Short Answer Type - 3 Marks Each)
6.
Simplify the following expression:
$(a + b)(2a - 3b + c) - (2a - 3b)c$
7.
Using the standard identity $(x+a)(x+b) = x^2 + (a+b)x + ab$, find the product:
$(4x + 5)(4x + 1)$
8.
Simplify the following expression using suitable identities:
$(3x - 5y)^2 + (3x + 5y)^2$
9.
If $\left(x + \frac{1}{x}\right) = 5$, find the value of $\left(x^2 + \frac{1}{x^2}\right)$.
10.
Evaluate $103 \times 104$ using a suitable algebraic identity.
SECTION C (15 Marks)(Long Answer Type - 5 Marks Each)
11.
(a) Subtract $3pq(p - q)$ from $2pq(p + q)$. (3 Marks)
(b) Show that $(3x + 7)^2 - 84x = (3x - 7)^2$. (2 Marks)
12.
(a) Simplify the expression using appropriate identities:
$(1.5x - 4y)(1.5x + 4y + 3) - 4.5x + 12y$ (3 Marks)
(b) Find the value of $x$ if $8x = 55^2 - 45^2$. (2 Marks)
13.
(a) If $a^2 + b^2 = 74$ and $ab = 35$, find the positive value of $(a + b)$. (3 Marks)
(b) Multiply $(a^2 - b^2)$ by $(a^2 + b^2)$. Evaluate the result for $a = 2$ and $b = 1$. (2 Marks)
