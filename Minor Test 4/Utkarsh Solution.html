[2 Marks] Q2. Factorize completely: 9x2 - 16y2
Expressing the terms as perfect squares:
= (3x)2 - (4y)2 (1 Mark)

Answer: (3x - 4y)(3x + 4y) (1 Mark)
[2 Marks] Q3. Rationalise the denominator of: 1 / (√7 - 2)
Multiply numerator and denominator by the conjugating factor (√7 + 2):
= [ 1 / (√7 - 2) ] × [ (√7 + 2) / (√7 + 2) ] (1 Mark)
= (√7 + 2) / [(√7)2 - (2)2]
= (√7 + 2) / (7 - 4)
Answer: (√7 + 2) / 3 (1 Mark)
[2 Marks] Q4. Expand using a suitable identity: (2x - y + z)2
Using identity: (a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca
Here a = 2x, b = -y, c = z (½ Mark)
= (2x)2 + (-y)2 + (z)2 + 2(2x)(-y) + 2(-y)(z) + 2(z)(2x) (1 Mark)
Answer: 4x2 + y2 + z2 - 4xy - 2yz + 4zx (½ Mark)
[2 Marks] Q5. Factorize: x2 + 7x + 12
We need two numbers that multiply to 12 and add to 7. The numbers are 4 and 3.
= x2 + 4x + 3x + 12 (1 Mark)
= x(x + 4) + 3(x + 4)
Answer: (x + 3)(x + 4) (1 Mark)

Take out the common bracket (x - 2):

[3 Marks] Q10. Find 'a' and 'b' if: (√3 - 1) / (√3 + 1) = a - b√3
Rationalise the LHS:
= [ (√3 - 1)(√3 - 1) ] / [ (√3 + 1)(√3 - 1) ] (1 Mark)
LHS = (√3 - 1)2 / (3 - 1)
LHS = (3 + 1 - 2√3) / 2 = (4 - 2√3) / 2 = 2 - √3 (1 Mark)
Equating LHS to RHS: 2 - 1√3 = a - b√3
Answer: a = 2, b = 1 (1 Mark)

Step 1: Find 1/x
1/x = 1 / (3 + 2√2)
Rationalising: = [ 1 / (3 + 2√2) ] × [ (3 - 2√2) / (3 - 2√2) ] (1 Mark)
= (3 - 2√2) / (9 - 8) = 3 - 2√2 (1 Mark)
Step 2: Find (x + 1/x)
x + 1/x = (3 + 2√2) + (3 - 2√2) = 6 (1 Mark)
Step 3: Square both sides
(x + 1/x)2 = 62
x2 + 1/x2 + 2 = 36 (1 Mark)
x2 + 1/x2 = 36 - 2
Answer: 34 (1 Mark)

[5 Marks] Q13. Simplify: 1/(1+√2) + 1/(√2+√3) + 1/(√3+√4)
Rationalise each term separately by multiplying by its conjugate:
Term 1: [ 1 / (√2 + 1) ] × [ (√2 - 1) / (√2 - 1) ] = (√2 - 1) / (2 - 1) = √2 - 1 (1.5 Marks)
Term 2: [ 1 / (√3 + √2) ] × [ (√3 - √2) / (√3 - √2) ] = (√3 - √2) / (3 - 2) = √3 - √2 (1.5 Marks)
Term 3: [ 1 / (√4 + √3) ] × [ (√4 - √3) / (√4 - √3) ] = (√4 - √3) / (4 - 3) = √4 - √3 = 2 - √3 (1 Mark)
Now add all the simplified terms:
= (√2 - 1) + (√3 - √2) + (2 - √3)
The √2 and √3 terms cancel out.
= -1 + 2
Answer: 1 (1 Mark)
[5 Marks] Q14. If a + b + c = 9 and ab + bc + ca = 26, calculate a3 + b3 + c3 - 3abc.
Step 1: Find (a2 + b2 + c2)
We know: (a + b + c)2 = a2 + b2 + c2 + 2(ab + bc + ca) (1 Mark)
92 = (a2 + b2 + c2) + 2(26)
81 = a2 + b2 + c2 + 52
a2 + b2 + c2 = 81 - 52 = 29 (1.5 Marks)
Step 2: Use the identity for cubes
a3 + b3 + c3 - 3abc = (a + b + c)(a2 + b2 + c2 - (ab + bc + ca)) (1 Mark)
Substituting the known values:
= 9 × (29 - 26) (1 Mark)
= 9 × 3
Answer: 27 (0.5 Mark)
[5 Marks] Q15. Factorize completely: x6 - y6
Treat this as a difference of two squares first:
= (x3)2 - (y3)2
= (x3 - y3)(x3 + y3) (2 Marks)
Now apply the difference of cubes identity to the first bracket:
x3 - y3 = (x - y)(x2 + xy + y2) (1 Mark)
Apply the sum of cubes identity to the second bracket:
x3 + y3 = (x + y)(x2 - xy + y2) (1 Mark)
Combine all factors:
Answer: (x - y)(x + y)(x2 + xy + y2)(x2 - xy + y2) (1 Mark)
