Expression: $(a + b)(2a - 3b + c) - (2a - 3b)c$
Expand the first part:
$= a(2a - 3b + c) + b(2a - 3b + c) - (2ac - 3bc)$ (1 Mark)
$= 2a^2 - 3ab + ac + 2ab - 3b^2 + bc - 2ac + 3bc$ (1 Mark)
Combine like terms:
$= 2a^2 - 3b^2 + (-3ab + 2ab) + (ac - 2ac) + (bc + 3bc)$
$= 2a^2 - 3b^2 - ab - ac + 4bc$ (1 Mark)

SECTION C (15 Marks)
Ans 11.
(a) Subtraction:
Expand expressions first:
Expression 1 (to subtract): $3pq(p - q) = 3p^2q - 3pq^2$ (1 Mark)
Expression 2 (from): $2pq(p + q) = 2p^2q + 2pq^2$
Now perform: (Expression 2) - (Expression 1)
$= (2p^2q + 2pq^2) - (3p^2q - 3pq^2)$ (1 Mark)
$= 2p^2q + 2pq^2 - 3p^2q + 3pq^2$
$= 5pq^2 - p^2q$ (1 Mark)

(b) Show that:
LHS $= (3x + 7)^2 - 84x$
$= 9x^2 + 2(3x)(7) + 49 - 84x$
$= 9x^2 + 42x + 49 - 84x$ (1 Mark)
$= 9x^2 - 42x + 49$
This can be rewritten as $(3x)^2 - 2(3x)(7) + 7^2$
$= (3x - 7)^2$ = RHS. Hence proved. (1 Mark)
Ans 12.
(a) Simplification:
Break the first term: $(1.5x - 4y) [ (1.5x + 4y) + 3 ]$
$= (1.5x - 4y)(1.5x + 4y) + 3(1.5x - 4y)$ (1 Mark)
Apply $(a-b)(a+b) = a^2 - b^2$ on the first part:
$= [(1.5x)^2 - (4y)^2] + 4.5x - 12y$
$= 2.25x^2 - 16y^2 + 4.5x - 12y$ (1 Mark)
Now add the rest of the original expression:
$= 2.25x^2 - 16y^2 + 4.5x - 12y - 4.5x + 12y$
$= 2.25x^2 - 16y^2$ (1 Mark)

(b) Find $x$:
$8x = 55^2 - 45^2$
Using $a^2 - b^2 = (a-b)(a+b)$
$8x = (55 - 45)(55 + 45)$ (1 Mark)
$8x = 10 \times 100 = 1000$
$x = \frac{1000}{8} = 125$ (1 Mark)
Ans 13.
(a) Value of $(a+b)$:
We know the identity: $(a+b)^2 = a^2 + b^2 + 2ab$ (1 Mark)
Substitute the given values ($a^2+b^2 = 74$ and $ab = 35$):
$(a+b)^2 = 74 + 2(35)$ (1 Mark)
$(a+b)^2 = 74 + 70 = 144$
Taking positive square root: $(a+b) = \sqrt{144} = 12$ (1 Mark)

(b) Multiply and evaluate:
$(a^2 - b^2)(a^2 + b^2)$
Using $(x-y)(x+y) = x^2 - y^2$, where $x = a^2, y = b^2$
$= (a^2)^2 - (b^2)^2 = a^4 - b^4$ (1 Mark)
Evaluate for $a = 2, b = 1$:
$= 2^4 - 1^4$
$= 16 - 1 = 15$ (1 Mark)
