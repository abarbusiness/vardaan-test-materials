Class: 8 ICSE Subject: Mathematics Time: 1 Hour Max. Marks: 50
QUESTION PAPER
1.
Find the square root of \(0.0004\).
2.
Check whether \(216\) is a perfect cube or not.
3.
Add: \((3a + 4b - 5c), (2a - 3b + 2c), (a + b + c)\).
4.
Smallest number to multiply \(72\) to make a perfect square?
5.
Subtract \((5x^2 - 3x + 2)\) from \((8x^2 + 4x - 5)\).
6.
Find \(\sqrt{1764}\) using the division method.
7.
Find \(\sqrt[3]{13824}\) by prime factorization.
8.
Multiply \((2x + 3y)\) by \((3x - 4y)\).
9.
Smallest number to divide \(392\) to get a perfect cube.
10.
If \(x + \frac{1}{x} = 5\), find \(x^2 + \frac{1}{x^2}\).
11.
Least 4-digit perfect square.
12.
Simplify \((x + 2)(x + 3) + (x - 2)(x - 3)\) for \(x = 4\).
13.
Prove \((a + b)^2 - (a - b)^2 = 4ab\); evaluate \(105^2 - 95^2\).
14.
Find the side of a cubical box with volume \(13.824 \text{ m}^3\).
DETAILED SOLUTION KEY

1. \(\sqrt{0.0004} = \sqrt{4/10000} = 0.02\)

2. Yes, \(216 = 6^3\).

3. \(3a+2a+a + 4b-3b+b - 5c+2c+c = 6a+2b-2c\)

4. \(72 = 2^3 \times 3^2\). Need factor of \(2\).

5. \((8x^2+4x-5) - (5x^2-3x+2) = 3x^2+7x-7\)

6. \(\sqrt{1764} = 42\)

7. \(\sqrt[3]{13824} = 24\)

8. \((2x+3y)(3x-4y) = 6x^2+xy-12y^2\)

9. \(392 = 2^3 \times 7^2\). Divide by \(7^2 = 49\).

10. \((x+1/x)^2 = 25 \rightarrow x^2+1/x^2 = 23\)

11. \(32^2 = 1024\)

12. \(2x^2 + 12\). For \(x=4, 2(16)+12=44\).

13. \(4ab\). \(105^2-95^2 = (10)(200) = 2000\).

14. \(s^3 = 13.824 \rightarrow s = 2.4 \text{ m}\).
