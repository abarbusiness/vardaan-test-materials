[(x + 5) + (x - 3)] / [(x - 3)(x + 5)] = 1/3
(2x + 2) / (x2 + 2x - 15) = 1/3
3(2x + 2) = x2 + 2x - 15 ⇒ 6x + 6 = x2 + 2x - 15
x2 - 4x - 21 = 0 ⇒ x2 - 7x + 3x - 21 = 0

[5 Marks]
Let the speed of the stream be x km/h.
Speed of boat upstream = (18 - x) km/h
Speed of boat downstream = (18 + x) km/h
Time taken to go upstream = 24 / (18 - x) hours
Time taken to go downstream = 24 / (18 + x) hours
According to question: [24 / (18 - x)] - [24 / (18 + x)] = 1
24 × [ (18 + x - 18 + x) / ((18 - x)(18 + x)) ] = 1
24(2x) / (324 - x2) = 1
48x = 324 - x2 ⇒ x2 + 48x - 324 = 0 (Formulated Quadratic Equation)
x2 + 54x - 6x - 324 = 0
x(x + 54) - 6(x + 54) = 0 ⇒ (x + 54)(x - 6) = 0
x = 6 or x = -54. Since speed cannot be negative, we reject -54.
Speed of the stream = 6 km/h.

(a) Let the two consecutive odd positive integers be x and (x + 2).
Sum of their squares = 290 ⇒ x2 + (x + 2)2 = 290
x2 + x2 + 4x + 4 = 290
2x2 + 4x - 286 = 0 ⇒ x2 + 2x - 143 = 0
x2 + 13x - 11x - 143 = 0 ⇒ x(x + 13) - 11(x + 13) = 0
(x + 13)(x - 11) = 0 ⇒ x = 11 or x = -13.
Since the integer must be positive, x = 11.
The two odd positive integers are 11 and 13.
(b) Equation: 1⁄x − 1⁄(x - 2) = 3
Taking LCM: [(x - 2) - x] / [x(x - 2)] = 3
-2 / (x2 - 2x) = 3
3(x2 - 2x) = -2 ⇒ 3x2 - 6x + 2 = 0
Using quadratic formula: x = [-b ± √(b2 - 4ac)] / 2a
x = [6 ± √((-6)2 - 4(3)(2))] / (2 × 3)
x = [6 ± √(36 - 24)] / 6 = [6 ± √12] / 6
x = [6 ± 2√3] / 6 = [3 ± √3] / 3
Roots are x = (3 + √3)/3 and x = (3 - √3)/3.
