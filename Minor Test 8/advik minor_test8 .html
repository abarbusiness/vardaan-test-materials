MINOR TEST 8
Class: 8 ICSE Subject: Mathematics Time: 1 Hour Max. Marks: 50
General Instructions:
  • All questions are compulsory.
  • The paper is divided into three sections: A (10 Marks), B (20 Marks), and C (20 Marks).
  • Show necessary working for numerical problems.
SECTION A (10 Marks)(5 Questions - 2 Marks Each)
1.
Find the square root of \(0.0004\).
2.
Check whether \(216\) is a perfect cube or not. Justify your answer.
3.
Add the algebraic expressions: \((3a + 4b - 5c)\), \((2a - 3b + 2c)\), and \((a + b + c)\).
4.
What is the smallest number by which \(72\) must be multiplied to make it a perfect square?
5.
Subtract \((5x^2 - 3x + 2)\) from \((8x^2 + 4x - 5)\).
SECTION B (20 Marks)(5 Questions - 4 Marks Each)
6.
Find the square root of \(1764\) using the division method.
7.
Find the cube root of \(13824\) by the prime factorization method.
8.
Multiply \((2x + 3y)\) by \((3x - 4y)\) and simplify the result.
9.
Find the smallest number by which \(392\) must be divided so that the quotient is a perfect cube.
10.
If \(x + \frac{1}{x} = 5\), find the value of \(x^2 + \frac{1}{x^2}\).
SECTION C (20 Marks)(4 Questions - 5 Marks Each)
11.
Find the least number of \(4\) digits which is a perfect square. Show the steps.
12.
Simplify the expression \((x + 2)(x + 3) + (x - 2)(x - 3)\) and find its value when \(x = 4\).
13.
Show that \((a + b)^2 - (a - b)^2 = 4ab\). Using this identity, evaluate \(105^2 - 95^2\).
14.
The volume of a cubical box is \(13.824\) cubic meters. Find the length of each side of the box.
