Detailed Solutions: Practice Test (Trigonometry)

Question 1 [2 Marks]
In \(\triangle ABC\), right-angled at \(B\), \(AB = 3\text{ cm}\) and \(BC = 4\text{ cm}\). The value of \(\sin A\) is:
Correct Option: (b) \(4/5\)
Solution:
In \(\triangle ABC\), right angled at \(B\). Base \(AB = 3\), Perpendicular \(BC = 4\).
Using Pythagoras theorem: \(AC^2 = AB^2 + BC^2\)
\(AC^2 = 3^2 + 4^2 = 9 + 16 = 25 \Rightarrow AC = 5\)
\(\sin A = \frac{\text{Opposite to } A}{\text{Hypotenuse}} = \frac{BC}{AC} = \frac{4}{5}\).
Question 2 [2 Marks]
The value of \(\tan 45^\circ\) is:
Correct Option: (b) \(1\)
Solution:
From the standard trigonometric table, the value of \(\tan 45^\circ\) is exactly \(1\).
Since \(\sin 45^\circ = 1/\sqrt{2}\) and \(\cos 45^\circ = 1/\sqrt{2}\), \(\tan 45^\circ = \frac{\sin 45^\circ}{\cos 45^\circ} = 1\).
Question 3 [2 Marks]
The exact value of \(\sin 30^\circ\) is:
Correct Option: (a) \(1/2\)
Solution:
From the standard trigonometric table, the value of \(\sin 30^\circ\) is \(1/2\).
Question 4 [2 Marks]
Evaluate \(\cos^2 30^\circ - \sin^2 30^\circ\):
Correct Option: (b) \(1/2\)
Solution:
We know that \(\cos 30^\circ = \frac{\sqrt{3}}{2}\) and \(\sin 30^\circ = \frac{1}{2}\).
\(\cos^2 30^\circ - \sin^2 30^\circ = \left(\frac{\sqrt{3}}{2}\right)^2 - \left(\frac{1}{2}\right)^2\)
\(= \frac{3}{4} - \frac{1}{4} = \frac{2}{4} = \frac{1}{2}\).
Question 5 [2 Marks]
If \(\sin \theta = \cos \theta\) for an acute angle \(\theta\), then \(\theta\) is equal to:
Correct Option: (b) \(45^\circ\)
Solution:
Given \(\sin \theta = \cos \theta\). Dividing both sides by \(\cos \theta\):
\(\frac{\sin \theta}{\cos \theta} = 1 \Rightarrow \tan \theta = 1\).
We know \(\tan 45^\circ = 1\), hence \(\theta = 45^\circ\).
Question 6 [2 Marks]
According to trigonometric identities, \(1 - \sin^2 \theta\) is identically equal to:
Correct Option: (b) \(\cos^2 \theta\)
Solution:
By the fundamental trigonometric identity: \(\sin^2 \theta + \cos^2 \theta = 1\).
Rearranging the terms, we get \(\cos^2 \theta = 1 - \sin^2 \theta\).
Question 7 [2 Marks]
The value of the expression \(\sin^2 60^\circ + \cos^2 60^\circ\) evaluates to:
Correct Option: (b) \(1\)
Solution:
Using the identity \(\sin^2 \theta + \cos^2 \theta = 1\).
Here \(\theta = 60^\circ\). Therefore, \(\sin^2 60^\circ + \cos^2 60^\circ = 1\).
(Alternatively: \((\sqrt{3}/2)^2 + (1/2)^2 = 3/4 + 1/4 = 1\)).
Question 8 [2 Marks]
If \(\tan A = 4/3\), find the value of \(\sin A\):
Correct Option: (a) \(4/5\)
Solution:
\(\tan A = \frac{\text{Perpendicular}}{\text{Base}} = \frac{4}{3}\).
Let Perpendicular = \(4k\), Base = \(3k\). Hypotenuse \(H = \sqrt{(4k)^2 + (3k)^2} = \sqrt{25k^2} = 5k\).
\(\sin A = \frac{\text{Perpendicular}}{\text{Hypotenuse}} = \frac{4k}{5k} = \frac{4}{5}\).

The numerical value of \(\sin 90^\circ\) is:

Solution:
From the standard trigonometric table, the value of \(\sin 90^\circ = 1\).
Question 10 [2 Marks]
What is the maximum value of \(\sin \theta\) for \(0^\circ \le \theta \le 90^\circ\)?
Correct Option: (c) \(1\)
Solution:
The value of \(\sin \theta\) increases from \(0\) to \(1\) as \(\theta\) goes from \(0^\circ\) to \(90^\circ\).
Hence, the maximum value is \(1\) (at \(\theta = 90^\circ\)).
Question 11 [2 Marks]
\(\tan \theta\) can be written in terms of \(\sin \theta\) and \(\cos \theta\) as:
Correct Option: (b) \(\frac{\sin\theta}{\cos\theta}\)
Solution:
By definition, \(\tan \theta = \frac{\text{Opposite}}{\text{Adjacent}}\).
Dividing numerator and denominator by Hypotenuse, we get \(\frac{\sin \theta}{\cos \theta}\).
Question 12 [2 Marks]
The value of \(\cos 0^\circ\) is:
Correct Option: (b) \(1\)
Solution:
From the trigonometric standard tables, \(\cos 0^\circ = 1\).
Question 13 [2 Marks]
Evaluate the expression \(9 \sec^2 A - 9 \tan^2 A\):
Correct Option: (b) \(9\)
Solution:
Taking \(9\) common: \(9(\sec^2 A - \tan^2 A)\).
We know the identity \(1 + \tan^2 A = \sec^2 A \Rightarrow \sec^2 A - \tan^2 A = 1\).
Therefore, \(9(1) = 9\).
Question 14 [2 Marks]
The expression \(\frac{1 + \tan^2 A}{1 + \cot^2 A}\) simplifies to:
Correct Option: (d) \(\tan^2 A\)
Solution:
Using identities: \(1 + \tan^2 A = \sec^2 A\) and \(1 + \cot^2 A = \operatorname{cosec}^2 A\).
The expression becomes \(\frac{\sec^2 A}{\operatorname{cosec}^2 A} = \frac{1/\cos^2 A}{1/\sin^2 A}\).
\(= \frac{\sin^2 A}{\cos^2 A} = \tan^2 A\).
Question 15 [2 Marks]
If \(\sin A = 1/2\), then the value of \(\cos A\) is:
Correct Option: (a) \(\sqrt{3}/2\)
Solution:
If \(\sin A = 1/2\), then angle \(A = 30^\circ\).
Therefore, \(\cos A = \cos 30^\circ = \frac{\sqrt{3}}{2}\).
(Or use \(\cos A = \sqrt{1 - \sin^2 A} = \sqrt{1 - 1/4} = \sqrt{3/4}\)).
Question 16 [2 Marks]
If \(\triangle PQR\) is right-angled at \(Q\), and \(\sin P = 5/13\), find \(\tan P\).
Correct Option: (a) \(5/12\)
Solution:
\(\sin P = \frac{\text{Perpendicular}}{\text{Hypotenuse}} = \frac{5}{13}\).
\(\text{Base} = \sqrt{13^2 - 5^2} = \sqrt{169 - 25} = \sqrt{144} = 12\).
\(\tan P = \frac{\text{Perpendicular}}{\text{Base}} = \frac{5}{12}\).
Question 17 [2 Marks]
Calculate the value of \(\frac{2 \tan 30^\circ}{1 + \tan^2 30^\circ}\):
Correct Option: (a) \(\sin 60^\circ\)
Solution:
Substitute \(\tan 30^\circ = \frac{1}{\sqrt{3}}\).
Numerator: \(2 \times \frac{1}{\sqrt{3}} = \frac{2}{\sqrt{3}}\).
Denominator: \(1 + \left(\frac{1}{\sqrt{3}}\right)^2 = 1 + \frac{1}{3} = \frac{4}{3}\).
Result: \(\frac{2/\sqrt{3}}{4/3} = \frac{2}{\sqrt{3}} \times \frac{3}{4} = \frac{\sqrt{3}}{2}\).
Since \(\sin 60^\circ = \frac{\sqrt{3}}{2}\), option (a) is correct.
Question 18 [2 Marks]
Find the value of \(\frac{1 - \tan^2 45^\circ}{1 + \tan^2 45^\circ}\):
Correct Option: (d) \(0\)
Solution:
Substitute \(\tan 45^\circ = 1\).
Expression becomes: \(\frac{1 - 1^2}{1 + 1^2} = \frac{1 - 1}{1 + 1} = \frac{0}{2} = 0\).
Question 19 [2 Marks]
\(\sin 2A = 2 \sin A\) is true when \(A\) is equal to:
Correct Option: (a) \(0^\circ\)
Solution:
Test the options. Let \(A = 0^\circ\).
LHS = \(\sin (2 \times 0^\circ) = \sin 0^\circ = 0\).
RHS = \(2 \sin 0^\circ = 2(0) = 0\). LHS = RHS.
Question 20 [2 Marks]
The expression \(\frac{2 \tan 30^\circ}{1 - \tan^2 30^\circ}\) is equal to:
Correct Option: (c) \(\tan 60^\circ\)
Solution:
Substitute \(\tan 30^\circ = 1/\sqrt{3}\).
Numerator: \(2/\sqrt{3}\). Denominator: \(1 - 1/3 = 2/3\).
Result: \(\frac{2/\sqrt{3}}{2/3} = \frac{2}{\sqrt{3}} \times \frac{3}{2} = \sqrt{3}\).
Since \(\tan 60^\circ = \sqrt{3}\), option (c) is correct.
Question 21 [2 Marks]
If \(\cos \theta = \frac{\sqrt{3}}{2}\), then the acute angle \(\theta\) is:

Solution:
From the standard trigonometric ratios, \(\cos 30^\circ = \frac{\sqrt{3}}{2}\).
Therefore, \(\theta = 30^\circ\).
Question 22 [2 Marks]
Evaluate: \(\sin 60^\circ \cos 30^\circ + \sin 30^\circ \cos 60^\circ\)
Correct Option: (c) \(1\)
Solution:
Substitute standard values: \(\left(\frac{\sqrt{3}}{2}\right)\left(\frac{\sqrt{3}}{2}\right) + \left(\frac{1}{2}\right)\left(\frac{1}{2}\right)\).
\(= \frac{3}{4} + \frac{1}{4} = \frac{4}{4} = 1\).
Question 23 [2 Marks]
What is the standard value of \(\sec^2 \theta - \tan^2 \theta\)?
Correct Option: (b) \(1\)
Solution:
By the standard Pythagorean trigonometric identity, \(1 + \tan^2 \theta = \sec^2 \theta\).
Rearranging yields: \(\sec^2 \theta - \tan^2 \theta = 1\).
Question 24 [2 Marks]
If \(\cot \theta = 7/8\), evaluate \(\frac{(1+\sin\theta)(1-\sin\theta)}{(1+\cos\theta)(1-\cos\theta)}\):
Correct Option: (a) \(49/64\)
Solution:
Simplify numerator using \((a+b)(a-b) = a^2-b^2\): \(1 - \sin^2 \theta = \cos^2 \theta\).
Simplify denominator similarly: \(1 - \cos^2 \theta = \sin^2 \theta\).
Expression becomes \(\frac{\cos^2 \theta}{\sin^2 \theta} = \cot^2 \theta\).
Given \(\cot \theta = 7/8\), so \(\cot^2 \theta = (7/8)^2 = 49/64\).
Question 25 [2 Marks]
If \(3 \cot A = 4\), find the value of \(\cos A\).
Correct Option: (b) \(4/5\)
Solution:
\(3 \cot A = 4 \Rightarrow \cot A = 4/3\).
Base = \(4\), Perpendicular = \(3\). Hypotenuse = \(\sqrt{4^2 + 3^2} = \sqrt{16 + 9} = 5\).
\(\cos A = \frac{\text{Base}}{\text{Hypotenuse}} = \frac{4}{5}\).
Question 26 [2 Marks]
The exact value of \(\operatorname{cosec} 45^\circ\) is:
Correct Option: (b) \(\sqrt{2}\)
Solution:
\(\operatorname{cosec} 45^\circ\) is the reciprocal of \(\sin 45^\circ\).
Since \(\sin 45^\circ = 1/\sqrt{2}\), \(\operatorname{cosec} 45^\circ = \sqrt{2}\).
Question 27 [2 Marks]
The exact value of \(\cot 60^\circ\) is:
Correct Option: (b) \(1/\sqrt{3}\)
Solution:
\(\cot 60^\circ = \frac{1}{\tan 60^\circ}\).
Since \(\tan 60^\circ = \sqrt{3}\), \(\cot 60^\circ = 1/\sqrt{3}\).
Question 28 [2 Marks]
If \(\sec \theta = 13/12\), calculate \(\sin \theta\).
Correct Option: (c) \(5/13\)
Solution:
\(\sec \theta = \frac{\text{Hypotenuse}}{\text{Base}} = \frac{13}{12}\).
\(\text{Perpendicular} = \sqrt{13^2 - 12^2} = \sqrt{169 - 144} = \sqrt{25} = 5\).
\(\sin \theta = \frac{\text{Perpendicular}}{\text{Hypotenuse}} = \frac{5}{13}\).
Question 29 [2 Marks]
What is the value of \((\sin 30^\circ + \cos 30^\circ) - (\sin 60^\circ + \cos 60^\circ)\)?
Correct Option: (b) \(0\)
Solution:
Substitute values: \((1/2 + \sqrt{3}/2) - (\sqrt{3}/2 + 1/2)\).
Notice both brackets yield the exact same sum. Thus, the difference is \(0\).
Question 30 [2 Marks]
Which of the following is equivalent to \(\sqrt{1 - \cos^2 A}\)?
Correct Option: (c) \(\sin A\)
Solution:
By the identity \(\sin^2 A + \cos^2 A = 1\), we have \(\sin^2 A = 1 - \cos^2 A\).
Taking the square root (for acute angle A), \(\sin A = \sqrt{1 - \cos^2 A}\).
Question 31 [2 Marks]
In a right triangle \(ABC\), right-angled at \(B\), if \(\tan A = 1\), find the value of \(2 \sin A \cos A\).
Correct Option: (b) \(1\)
Solution:
\(\tan A = 1 \Rightarrow A = 45^\circ\).
\(2 \sin A \cos A = 2 \sin 45^\circ \cos 45^\circ = 2 \left(\frac{1}{\sqrt{2}}\right)\left(\frac{1}{\sqrt{2}}\right)\).
\(= 2 \times \frac{1}{2} = 1\).
Question 32 [2 Marks]
If \(\triangle ABC\) is right-angled at \(C\), then the value of \(\cos(A+B)\) is:
Correct Option: (a) \(0\)
Solution:
In \(\triangle ABC\), \(A + B + C = 180^\circ\). Given \(C = 90^\circ\).
So, \(A + B = 180^\circ - 90^\circ = 90^\circ\).
Therefore, \(\cos(A+B) = \cos(90^\circ) = 0\).
Question 33 [2 Marks]
Given \(15 \cot A = 8\), find \(\sin A\).
Correct Option: (b) \(15/17\)
Solution:
\(\cot A = 8/15\). Base = \(8\), Perpendicular = \(15\).
Hypotenuse = \(\sqrt{8^2 + 15^2} = \sqrt{64 + 225} = \sqrt{289} = 17\).
\(\sin A = \frac{\text{Perpendicular}}{\text{Hypotenuse}} = \frac{15}{17}\).
Question 34 [2 Marks]
The value of \(\sec A\) or \(\operatorname{cosec} A\) is always:
Correct Option: (c) \(\ge 1\)
Solution:
\(\sec A = \frac{\text{Hypotenuse}}{\text{Base}}\) and \(\operatorname{cosec} A = \frac{\text{Hypotenuse}}{\text{Perpendicular}}\).
Since Hypotenuse is the longest side, the ratio is always greater than or equal to \(1\).
Question 35 [2 Marks]
Evaluate the expression \(\sin^2 45^\circ + \cos^2 45^\circ\):
Correct Option: (b) \(1\)
Solution:
By standard identity, \(\sin^2 \theta + \cos^2 \theta = 1\).
Or by values: \((1/\sqrt{2})^2 + (1/\sqrt{2})^2 = 1/2 + 1/2 = 1\).
Question 36 [2 Marks]
Which of the following is exactly equal to \(\cos 30^\circ\)?
Correct Option: (a) \(\sin 60^\circ\)
Solution:
\(\cos 30^\circ = \sqrt{3}/2\).
Checking options, \(\sin 60^\circ = \sqrt{3}/2\). They are complementary ratios.
Question 37 [2 Marks]
Express \(\operatorname{cosec}^2 \theta - 1\) in terms of another trigonometric ratio:
Correct Option: (c) \(\cot^2 \theta\)
Solution:
Using the standard identity: \(1 + \cot^2 \theta = \operatorname{cosec}^2 \theta\).
Rearranging yields: \(\operatorname{cosec}^2 \theta - 1 = \cot^2 \theta\).
Question 38 [2 Marks]
The simplification of \((\sec A + \tan A)(1 - \sin A)\) yields:
Correct Option: (d) \(\cos A\)
Solution:
Convert to sine and cosine: \((\frac{1}{\cos A} + \frac{\sin A}{\cos A})(1 - \sin A)\).
\(= \left(\frac{1 + \sin A}{\cos A}\right) (1 - \sin A) = \frac{(1 + \sin A)(1 - \sin A)}{\cos A}\).
\(= \frac{1 - \sin^2 A}{\cos A} = \frac{\cos^2 A}{\cos A} = \cos A\).
Question 39 [2 Marks]
If \(\sin \theta = x\) and \(\sec \theta = y\), then \(\tan \theta\) is equal to:
Correct Option: (c) \(xy\)
Solution:
We know \(\tan \theta = \frac{\sin \theta}{\cos \theta}\).
Since \(\sec \theta = 1/\cos \theta\), we can write \(\tan \theta = \sin \theta \times \sec \theta\).
Substitute given values: \(\tan \theta = x \times y = xy\).
Question 40 [2 Marks]
The expression \(\tan^2 \theta \cdot \cos^2 \theta\) simplifies to:
Correct Option: (a) \(\sin^2 \theta\)
Solution:
We know \(\tan \theta = \frac{\sin \theta}{\cos \theta}\). Therefore, \(\tan^2 \theta = \frac{\sin^2 \theta}{\cos^2 \theta}\).
Expression: \(\frac{\sin^2 \theta}{\cos^2 \theta} \times \cos^2 \theta\).
The \(\cos^2 \theta\) terms cancel out, leaving \(\sin^2 \theta\).
